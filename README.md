- Ubuntu 20.04
- ROS noetic

## 下载这个仓库

请使用git clone进行下载，而非直接点击download zip．如不了解git的基本操作，请先学一下．

请思考：应该把这个仓库的文件放在什么地方？
* 提示1：请参考ROS的工作空间、包的概念．
* 提示2：这里的每个assignment下面都是若干个ROS包．

这一点若有疑问，请在群里提问．

## 任务
